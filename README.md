# Noders.com App :D

##Instalación

1. Clonar el repo
2. Instalar webpack globalmente
  * npm install webpack -g
3. Instalar dependencias
  * npm install && bower install
4. Descargar e instalar la API
  * https://github.com/Noders/Noders-API
5. Correr la app! 
  * npm run-script dev
